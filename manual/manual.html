1Xautopost คืออะไร

Xautopost คือโปรแกรมเดสก์ท็อป (ติดตั้งบนคอมพิวเตอร์) สำหรับ จัดการโพสต์ X (Twitter เดิม) หลายบัญชีพร้อมกัน โดยที่ไม่ต้องเข้าออกแต่ละบัญชีเอง ระบบจะหมุนเวียนโพสต์ตามเวลาและความถี่ ที่คุณตั้งให้แต่ละบัญชี

สิ่งที่ทำได้

  • หลายบัญชีในที่เดียว เพิ่มกี่บัญชีก็ได้ ระบบจัดการให้ทั้งหมดจากหน้าเดียว
  • โพสต์อัตโนมัติด้วย AI ใช้ OpenAI หรือ Google Gemini เขียนทวีตให้ตามสไตล์ที่คุณกำหนด
  • โพสต์อัตโนมัติแบบเขียนเอง ไม่อยากใช้ AI ก็ได้ พิมพ์ข้อความเองหลายๆ แบบ ระบบจะสุ่มหยิบมาโพสต์
  • ตั้งช่วงเวลาและความถี่ เช่น โพสต์ระหว่าง 9 โมงเช้า ถึง 22 โมงเย็น ทุก 30 นาทีถึง 2 ชั่วโมง ไม่เกิน 10 ครั้งต่อวัน
  • ปลอดภัย เก็บในเครื่อง ข้อมูลทุกอย่าง (cookie, API key, รหัสผ่าน) เก็บไว้ในคอมพิวเตอร์คุณเอง เข้ารหัสครบ
เหมาะกับใคร: ผู้สร้างคอนเทนต์, เอเจนซี่ Social Media, คนทำหลายแบรนด์ ที่อยากเปิดบัญชี X หลายตัวและให้โพสต์ตามสไตล์เฉพาะของแต่ละบัญชี
สิ่งที่ต้องเตรียมก่อนเริ่ม:
  1. คอมพิวเตอร์ Mac (M1 ขึ้นไป) หรือ Windows 10/11
  2. Google Chrome ติดตั้งในเครื่อง (จำเป็น)
  3. บัญชี X ที่ต้องการให้ระบบโพสต์ พร้อมรหัสผ่าน
  4. API Key ของ OpenAI หรือ Gemini (เฉพาะถ้าจะใช้ AI)

2ดาวน์โหลดและติดตั้ง

ขั้นที่ 1 · เปิดหน้าดาวน์โหลด

เปิดเบราว์เซอร์ ไปที่

https://mergedryio2.github.io/Xautopost/

หน้าเว็บจะตรวจระบบของคุณอัตโนมัติ และเน้นปุ่มดาวน์โหลดที่เหมาะสม (Mac Apple Silicon, Mac Intel, หรือ Windows) มีกรอบเขียนว่า "แนะนำสำหรับเครื่องคุณ"

ขั้นที่ 2 · กดดาวน์โหลด

  • ถ้าเป็น Mac จะได้ไฟล์ .dmg (ขนาด ~200 MB)
  • ถ้าเป็น Windows จะได้ไฟล์ .exe (ขนาด ~160 MB)

ขั้นที่ 3 · ติดตั้ง

สำหรับ Mac

  1. เปิดไฟล์ .dmg ที่ดาวน์โหลดมา
  2. ลาก icon Xautopost ลงในโฟลเดอร์ Applications
  3. เปิดโปรแกรมจาก Launchpad หรือ Applications
ถ้า Mac แสดงข้อความ "แอปนี้เสียหาย" หรือ "ไม่สามารถเปิดได้": เปิด Terminal (Spotlight ค้นว่า "Terminal") แล้วพิมพ์คำสั่งนี้แล้วกด Enter:

xattr -cr /Applications/Xautopost.app

จากนั้นเปิดแอปใหม่ ปกติเลย (เกิดเพราะแอปยังไม่ได้ลงทะเบียนกับ Apple)

สำหรับ Windows

  1. เปิดไฟล์ .exe ที่ดาวน์โหลดมา
  2. ถ้าหน้า Windows SmartScreen ขึ้นมาเตือน ให้คลิก "More info" แล้วคลิก "Run anyway"
  3. เลือกโฟลเดอร์ติดตั้ง (ปกติ C:\Program Files\Xautopost)
  4. กด Install · เสร็จแล้วเปิดได้จาก Start Menu

3เริ่มใช้งานครั้งแรก

เมื่อเปิดแอปครั้งแรก จะเจอหน้า สร้างโปรไฟล์ ก่อน · โปรไฟล์คือ "บัญชีผู้ใช้ระบบ" ของคุณ ใช้สำหรับล็อกอินเข้าแอป ไม่เกี่ยวกับบัญชี X

ขั้นที่ 1 · กรอกข้อมูล

  1. ชื่อโปรไฟล์ · ตั้งเป็นชื่อเล่น เช่น "Mint", "Workspace 1", "งานหลัก" ฯลฯ ใช้เพื่อจำว่าเป็นโปรไฟล์ใคร
  2. รหัสผ่าน · ตั้งรหัสผ่านอย่างน้อย 4 ตัวอักษร จำให้ดี เพราะใช้เปิดแอปครั้งต่อๆ ไป
  3. สีโปรไฟล์ · เลือกสีที่ชอบ ใช้เป็นสีตัวอักษรย่อในแอป

ขั้นที่ 2 · กด "สร้างโปรไฟล์"

ระบบจะพาเข้า Setup Wizard 4 ขั้นตอน · แนะนำให้ทำตามไป ใช้เวลาไม่เกิน 3 นาที

สร้างได้หลายโปรไฟล์ · ถ้าใช้คอมเครื่องเดียวกันแต่อยากแยก งานคนละ workspace เช่นบัญชีลูกค้า A กับบัญชีของตัวเอง สร้างได้เลย แต่ละโปรไฟล์มีบัญชี X / API key / สไตล์การเขียน แยกกันทั้งหมด

4Setup Wizard · 4 ขั้นตอนเตรียมระบบ

Wizard เป็นตัวช่วยเริ่มต้นใช้งาน · ผ่านครบจะเริ่มโพสต์อัตโนมัติได้ทันที · ข้ามข้ามไปทำทีหลังได้ทุกขั้น

ขั้นที่ 1 · ทักทาย

แค่กด "เริ่มกันเลย →" เพื่อไปต่อ

ขั้นที่ 2 · เชื่อม AI

เลือกว่าจะใช้ OpenAI (GPT) หรือ Google Gemini · กดการ์ดที่ต้องการแล้วใส่ API Key

วิธีหา API Key:
  • OpenAI ไปที่ platform.openai.com/api-keys · กด Create new secret key · copy มาวาง · เริ่มต้นด้วย sk-...
  • Gemini ไปที่ aistudio.google.com/apikey · กด Create API key · copy มาวาง · เริ่มต้นด้วย AIza...

กรอก Key แล้วกด "เชื่อม AI →" · ระบบจะเก็บกุญแจ เข้ารหัสในเครื่อง

ไม่อยากใช้ AI? · กด "ข้ามขั้นนี้" แล้วใช้สไตล์ "เขียนเอง" ในขั้นถัดไป (ดูส่วน 7)

ขั้นที่ 3 · ตั้งสไตล์การเขียน

เลือก template สำเร็จรูป 1 อันจากที่มีให้ เช่น "คำคมให้กำลังใจ", "ชวนกินอะไร", "ข่าวเทคย่อย", "มุกเบาๆ", "ก่อนนอน" หรือ "AI ตามใจ" (เขียน prompt ให้ AI เอง)

หรือเลือก "📝 เขียนเอง" ที่ใช้ข้อความที่เราพิมพ์เอง ไม่เรียก AI เลย

กด "ใช้สไตล์นี้ →" ระบบจะบันทึก

ขั้นที่ 4 · เพิ่มบัญชี X แรก

กด "เปิด browser เลย →" · จะมีหน้าต่าง Chrome เปิดขึ้นมา ให้คุณ login เข้า X ตามปกติ

สำคัญ: login ด้วย email + รหัสผ่าน ของบัญชี X โดยตรง · อย่ากด "Sign in with Google" เพราะ Google จะตรวจจับว่าเป็น automated browser และบล็อก

เมื่อ login ผ่าน ระบบจะปิด browser อัตโนมัติ และบันทึก session เข้ารหัสในเครื่อง · เห็นข้อความ "login สำเร็จ" แสดงว่าใช้ได้แล้ว

ขั้นที่ 5 · เรียบร้อย

Wizard จะแสดง checklist ของสิ่งที่ตั้งค่าเสร็จ · กด "ไปยังหน้าหลัก" เพื่อเริ่มใช้งาน

  • "ออกจากระบบ" · กลับไปหน้า login (ข้อมูลทั้งหมดยังอยู่ในเครื่อง · login เข้ามาใหม่ได้ตามเดิม)

6เพิ่มบัญชี X (login ผ่าน browser)

ถ้ายังไม่ได้ทำในตอน Setup Wizard หรือต้องการเพิ่มบัญชีอีก:

  1. ไปที่แท็บ 🎭 บัญชี
  2. กดปุ่ม "+ เพิ่มบัญชี X" มุมขวาบน
  3. อ่านข้อความเตือน · กด "เปิด browser เลย"
  4. หน้าต่าง Chrome จะเปิดมาที่ x.com · login ด้วย email + รหัสผ่าน (ไม่ใช่ Sign in with Google)
  5. พอ login ผ่าน หน้า X home จะโผล่ขึ้นมา · ระบบบันทึกอัตโนมัติ ปิด browser ให้
  6. กลับมาที่แอป · บัญชีจะปรากฏในรายการ พร้อมข้อความ "login สำเร็จ"
มี 2FA? · ใส่ OTP/รหัสยืนยันใน browser ตามปกติได้เลย ระบบจะรอจนคุณ login เสร็จ · หากต้อง verify email ก็ทำได้
ขีดจำกัด · ระบบรอ login ได้สูงสุด 5 นาที ถ้าใช้เกินกว่านั้นจะยกเลิก · ลองอีกรอบได้

7สไตล์การเขียน · AI vs เขียนเอง

แต่ละบัญชี X จะผูกกับ "สไตล์การเขียน" 1 อัน ใช้กำหนดว่าจะโพสต์อะไรในแต่ละครั้ง · มี 2 รูปแบบใหญ่:

🤖 สไตล์ AI

ใช้ OpenAI หรือ Gemini สร้างทวีตให้ใหม่ทุกครั้ง · คุณบรรยายว่าอยาก ให้ AI เขียนแนวไหน เช่น "คำคมให้กำลังใจ ภาษาไทย ไม่เกิน 240 ตัวอักษร"

ข้อดี

  • ทวีตใหม่ทุกครั้ง ไม่ซ้ำ
  • ปรับสไตล์ได้ละเอียด

ข้อเสีย

  • ต้องมี API key (มีค่าใช้จ่ายตามจริง · ปกติประมาณโพสต์ละ 1-2 บาท)
  • คุณภาพขึ้นกับ prompt ที่เขียนให้ AI

📝 สไตล์เขียนเอง

คุณพิมพ์ข้อความที่จะโพสต์เอง ใส่หลายๆ ข้อความก็ได้ · ระบบจะสุ่ม หยิบมาโพสต์ครั้งละ 1 ข้อความ

วิธีใส่หลายข้อความ: คั่นแต่ละข้อความด้วยบรรทัด --- (ขีด 3 ครั้ง) เช่น 
สวัสดีตอนเช้าค่ะ ขอให้วันนี้เป็นวันที่ดี
---
เช้านี้กาแฟแก้วโปรดต้องมา
---
วันใหม่ ลองอะไรใหม่ๆ ดูบ้าง

ข้อดี

  • ไม่ต้องเชื่อม AI · ไม่มีค่าใช้จ่าย
  • ควบคุมเนื้อหาได้ทุกตัวอักษร

ข้อเสีย

  • X ไม่อนุญาตให้โพสต์ข้อความเดียวกันซ้ำใน 24 ชม. ถ้าใส่ข้อความน้อยจะหยิบซ้ำและถูก reject
  • ต้องอัพเดตข้อความเองเป็นระยะ ไม่งั้นคนอ่านเดาง่าย

🎲 ระบบกันโพสต์ซ้ำอัตโนมัติ

สไตล์ "เขียนเอง" มี toggle "ใส่อิโมจิสุ่มท้ายแต่ละโพสต์" เปิดไว้ตามค่า default · ระบบจะหยิบ emoji 1 ตัวจากชุด 38 ตัวต่อท้ายข้อความ ทุกครั้ง ทำให้แต่ละโพสต์ไม่เหมือนกัน 100% (เช่น "สวัสดี ✨", "สวัสดี 🌸", "สวัสดี 💛")

ใส่ข้อความเท่าไหร่ดี? · ขั้นต่ำ 3-5 ข้อความ ถ้าเปิดอิโมจิสุ่มไว้ · หรือ 10+ ข้อความ ถ้าปิดอิโมจิสุ่ม

วิธีตั้งสไตล์ให้บัญชี

  1. ไปแท็บ 🎭 บัญชี
  2. ที่การ์ดของบัญชีที่ต้องการ ในแถบ "✦ ยังไม่ได้ตั้งสไตล์การเขียน" กดปุ่ม "ตั้งสไตล์"
  3. จะเปิดหน้าเลือกสไตล์ · เลือกจาก "สไตล์ที่มีอยู่แล้ว" หรือสร้างใหม่จาก template
  4. กด → สไตล์จะผูกกับบัญชีนั้น

8เปิดโพสต์อัตโนมัติ

เมื่อมี บัญชี X + สไตล์การเขียน ครบแล้ว ก็เปิดให้ระบบโพสต์อัตโนมัติได้

วิธีเปิด

  1. ไปแท็บ 🎭 บัญชี
  2. ที่การ์ดของบัญชีที่ต้องการ ดูสวิตช์ทางขวา · ตอนนี้น่าจะเขียนว่า "หยุดอยู่" สีเทา
  3. กดสวิตช์ · มันจะกลายเป็นสีเขียวมินต์เขียนว่า "อัตโนมัติ"
  4. เห็นแถบเขียวอ่อนที่บอกว่า "✓ กำลังโพสต์อัตโนมัติ · ระหว่าง 09:00-22:00 ทุก 30-120 นาที ไม่เกิน 10 ครั้งต่อวัน" → เรียบร้อย
ถ้าสวิตช์กดไม่ได้ · แสดงว่ายังไม่ได้ตั้งสไตล์การเขียน ตั้งให้ก่อนถึงจะเปิดได้

วิธีปิด

กดสวิตช์อีกครั้ง · กลายเป็นสีเทา "หยุดอยู่" · ระบบจะหยุดโพสต์ทันที แต่ session login + ข้อมูลต่างๆ ยังเก็บไว้ครบ

เปิดทีละบัญชีหรือทั้งหมด · เปิด-ปิดได้อิสระต่อบัญชี เช่น เปิด @account1 อย่างเดียว ส่วน @account2 หยุด

9ดูสถานะแบบ realtime

ที่ 🏠 หน้าหลัก · "บัญชีที่กำลังหมุนเวียน" จะมีสติกเกอร์สีต่างๆ บอกสถานะของแต่ละบัญชี อัพเดททุก 5 วินาที:

📮 กำลังโพสต์อยู่ตอนนี้

ระบบเปิด browser โพสต์อยู่ · จะเห็นหน้าต่าง Chrome เด้งขึ้นมา

🟢 พร้อมโพสต์ทุกเมื่อ · 5/10 วันนี้

พร้อมโพสต์แล้ว · รอ tick ถัดไปของ scheduler

⏱ รออีก 12 นาที – 1 ชม.

เพิ่งโพสต์ไป กำลังนับเวลารอช่วงต่อไป

🌙 พักนอกช่วงเวลา · เริ่มอีกครั้ง 09:00

ตอนนี้นอกช่วงเวลาที่ตั้งไว้ (เช่น เปิดเฉพาะ 9 โมง-22 โมง)

🛑 ครบโควต้าวันนี้ (10/10) · พรุ่งนี้

โพสต์ครบจำนวนสูงสุดที่ตั้งไว้ · เริ่มใหม่อัตโนมัติเมื่อข้ามวัน

⚠️ ยังไม่ได้ตั้งสไตล์การเขียน

ต้องตั้งสไตล์ก่อน บัญชีถึงจะเริ่มโพสต์ได้

บัญชีที่ "กำลังโพสต์อยู่ตอนนี้" การ์ดจะมีขอบสีเขียวมินต์ + ป้ายเต้นเป็นจังหวะ · เห็นชัดว่าตอนนี้กำลังทำงาน

มีบัญชีเกิน 4? · หน้าหลักจะแสดง 4 บัญชีแรก (เรียง ตามที่กำลังจะโพสต์ก่อน) · กดปุ่ม "ดูทั้งหมด" เพื่อเปิด modal ใหญ่ดูครบทุกบัญชี

10ทดลองโพสต์ทันที

ถ้าอยากทดสอบว่าระบบโพสต์ได้จริงไหม โดยไม่ต้องรอ scheduler:

  1. ไปแท็บ 🎭 บัญชี
  2. กดปุ่ม "ทดลองโพสต์" มุมขวาบน
  3. เลือกบัญชีที่จะให้โพสต์ (ติ๊กได้หลายบัญชี · กด "เลือกทั้งหมด" ก็ได้)
  4. เลือกสไตล์การเขียน · กด "✨ ให้ AI ลองเขียน" (สำหรับ AI mode) หรือ "🎲 หยิบข้อความสุ่ม" (สำหรับ manual mode) เพื่อให้ระบบใส่ข้อความให้
  5. ตรวจสอบเนื้อหาในกล่อง · แก้ไขได้ตามต้องการ · ความยาวไม่เกิน 280 ตัวอักษร
  6. กด "โพสต์เลย (X บัญชี)"

ระบบจะเปิด Chrome ทีละบัญชี โพสต์ทีละบัญชี (sequential) เห็น progress bar real-time · ผลลัพธ์แสดงด้านล่าง:

  • 🟢 สำเร็จ · โพสต์ขึ้น X แล้ว
  • 🔴 ไม่สำเร็จ · พร้อมเหตุผล (เช่น session หมดอายุ, content ซ้ำ)

11ตั้งค่าเวลาและความถี่

แต่ละบัญชีตั้งเวลาเป็นของตัวเอง · ที่การ์ดบัญชี กดปุ่ม "ตั้งค่าเวลา"

ช่วงเวลาที่จะโพสต์

เลือกจาก preset 4 อัน หรือ "กำหนดเอง":

  • ตลอดวัน · 00:00–23:00 (โพสต์ได้ทุกชั่วโมง)
  • เช้า-ค่ำ · 07:00–22:00 (เช้าตื่นถึงดึก)
  • กลางวัน · 09:00–18:00 (เวลาทำงาน)
  • เย็น-ดึก · 17:00–23:00 (หลังเลิกงาน)
  • กำหนดเอง · เลือกชั่วโมงเริ่มและหยุดเอง

โพสต์ได้สูงสุดต่อวัน

ตัวเลขกำหนด จำนวนครั้งสูงสุดต่อ 24 ชม. ค่า default คือ 10 · เมื่อครบจำนวนระบบจะหยุดบัญชีนั้น เริ่มใหม่อัตโนมัติ เมื่อข้ามเที่ยงคืน

เว้นช่วงระหว่างโพสต์

กำหนด ต่ำสุด และ สูงสุด เป็นนาที · ระบบจะสุ่มเวลาในช่วงนี้ทุกครั้ง ทำให้ดูเป็นธรรมชาติ ไม่เหมือน bot โพสต์ตรงเวลา

ขั้นต่ำ 5 นาที · X จะตรวจจับ pattern ของ bot ถ้าโพสต์ถี่กว่านี้ · ตั้งสูงๆ ปลอดภัยกว่า เช่น 30-120 นาที
คำแนะนำเริ่มต้น:
  • ช่วงเวลา · เช้า-ค่ำ (07:00–22:00)
  • โพสต์สูงสุด · 8-12 ครั้งต่อวัน
  • เว้นช่วง · ต่ำสุด 60 นาที สูงสุด 240 นาที

12แก้ปัญหาที่พบบ่อย

โพสต์ไม่สำเร็จ · "X ไม่ได้รับโพสต์"

เกิดได้หลายสาเหตุ:

  • เนื้อหาซ้ำ (X reject duplicate content ภายใน 24 ชม.) · ใช้สไตล์เขียนเองเปิด "อิโมจิสุ่ม" ไว้, หรือเพิ่มข้อความหลายๆ แบบ
  • ติด rate limit ชั่วคราว · ลดความถี่ลง · เพิ่มเวลาเว้น · พักไป 1-2 ชั่วโมง
  • บัญชีถูกจำกัด · X อาจล็อกบัญชีให้ verify ตัวตน · ลอง login ตรงๆ ที่ x.com ดูว่ามี popup ขอ verify หรือไม่

โพสต์ไม่สำเร็จ · "session หมดอายุ"

Cookies ที่บันทึกไว้หมดอายุ (ปกติ X จะให้ใช้ ~30 วัน)

  1. ลบบัญชีออกจากแอป (ปุ่ม "ลบบัญชี")
  2. เพิ่มบัญชีใหม่ login เข้าใหม่
  3. ตั้งสไตล์ + เปิด rotation อีกครั้ง

หน้า Chrome เปิดมาแล้ว Google block

คุณคงกด "Sign in with Google" · Google detect automated browser แล้ว block · กลับไปกด "ใช้ email" หรือ "Use another account" แล้ว login ด้วย email/รหัสผ่าน X โดยตรง

AI generate ล้มเหลว

  • เช็ค API key ยังใช้ได้อยู่ไหม (ไม่หมดอายุ, ไม่หมด credit)
  • ลองเปลี่ยนรุ่นโมเดล · กด "ขั้นสูง" ในฟอร์มสไตล์
  • ตรวจ prompt · บางครั้ง prompt ยาวเกินทำให้ AI งง

"missing X-Operator-Id header"

ถ้าเจอ error นี้ · ลอง ออกจากระบบ แล้ว login เข้าใหม่ · ปกติเกิดเมื่อ session ของแอปไม่ sync กัน

แอปไม่เปิดเลย / ค้าง

  1. ปิดแอปสนิท (Mac: Cmd+Q · Windows: Right click taskbar → Quit)
  2. เปิดใหม่
  3. ถ้ายังค้าง · ลบแอปแล้วติดตั้งใหม่ (ข้อมูลในเครื่องไม่หาย เพราะอยู่ คนละตำแหน่งกับโปรแกรม)

13เคล็ดลับใช้ระบบให้ได้ผล

กันโดน X ระงับบัญชี

  • อย่าตั้งความถี่ถี่เกิน · 30-120 นาทีปลอดภัย · ต่ำกว่า 15 นาทีเสี่ยง
  • เปิด "อิโมจิสุ่ม" สำหรับสไตล์เขียนเอง · กัน duplicate detection
  • ไม่ต้องโพสต์ 24 ชม. · ตั้งช่วงเวลาเหมือนคนใช้จริง · เช่น 7:00-23:00
  • เริ่มน้อยๆ ก่อน · บัญชีใหม่ตั้ง daily limit 5 ครั้ง · ค่อยเพิ่มเป็น 10-15 หลัง 1-2 สัปดาห์

เลือก AI ตัวไหนดี

  • OpenAI GPT-4o-mini · ราคาถูก คุณภาพดีพอ ภาษาไทยใช้ได้ · เหมาะกับการใช้ระยะยาว
  • Google Gemini 2.5 Flash · ฟรีระดับหนึ่ง · ความเร็วเท่ากัน · บางครั้งภาษาไทยติดๆ ขัดๆ

เขียน AI prompt อย่างไรให้ได้ผล

  1. บอกบทบาท · "คุณเป็น content creator ที่..."
  2. กำหนดความยาว · "ความยาวไม่เกิน 240 ตัวอักษร"
  3. กำหนดภาษา · "เขียนเป็นภาษาไทย"
  4. ห้ามชัด · "ห้ามใส่ # · ห้ามใช้ emoji เกิน 2 ตัว"
  5. ขอความหลากหลาย · "ทุกครั้งให้แตกต่างกัน หลีกเลี่ยงการซ้ำเดิม"
ตัวอย่าง prompt ที่ใช้ได้ดี:

"คุณเป็นคนเขียนคำคมให้กำลังใจคนไทยบน X · เขียนเป็นภาษาไทย · ความยาวไม่เกิน 240 ตัวอักษร · น้ำเสียงอบอุ่นจริงใจ · ห้ามใช้ภาษาตลาด · ห้ามใส่ # หรือ emoji เกิน 2 ตัว · ทุกครั้งให้สร้างคำคมที่แตกต่างจากครั้งก่อน"

เก็บประวัติการโพสต์

ทุกความพยายามโพสต์ (สำเร็จ/ล้มเหลว/ข้าม) ถูกบันทึกไว้ · ดูได้ที่ ⚙️ ตั้งค่า → "📜 ประวัติการโพสต์ทั้งหมด" · กรองตามบัญชีและสถานะได้

ใช้ proxy?
