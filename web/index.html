หลายบัญชี · รวมในที่เดียว

AI สร้างเนื้อหา

เชื่อม OpenAI หรือ Gemini ตั้ง prompt ให้ AI สร้างทวีตตามสไตล์ที่ต้องการ

หมุนเวียนอัตโนมัติ

ตั้งช่วงเวลาเปิด/ปิด, daily limit, rotation interval — ระบบจัดเองให้

ปลอดภัย · เก็บในเครื่อง

Cookies, API keys, proxy creds เข้ารหัส AES-GCM master key อยู่ใน OS Keychain

ความต้องการระบบ: ต้องมี Google Chrome ติดตั้งในเครื่อง (Playwright ใช้ Chrome เพื่อโพสต์) — Mac 11+ (Big Sur) หรือ Windows 10+

หมายเหตุ: แอปยังไม่ได้ code sign — Mac อาจขึ้น "App is damaged" ให้ใช้คำสั่ง xattr -cr /Applications/Xautopost.app หลังลากเข้า Applications · Windows อาจขึ้น SmartScreen ให้คลิก "More info" → "Run anyway"
